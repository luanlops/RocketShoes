## Instalação e uso

```bash
# Clone this repository
$ git clone https://github.com/luanlops/RocketShoes
# Go into the repository
$ cd rocketshoes
# Install dependencies
$ yarn install
# Run fake api
$ yarn server
# Run the app
$ yarn start
```
